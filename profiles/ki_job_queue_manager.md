# KI Job Queue Manager Profil

- Zweck: Lokale KI-Auftraege kontrolliert annehmen, priorisieren und abarbeiten
- Eingaben: Auftragsbeschreibung, Typ, Prioritaet, optionaler Startbefehl
- Ausgaben: Queue-Eintraege, Statusberichte, Logdateien, Fehlerdateien
- Leitplanken: Ressourcen vor jedem Start pruefen, Ueberlastung vermeiden, Queue persistent halten
