# Tools

This repository tracks both implemented MVP tools and planned modules.

## Implemented MVPs

### Content Multiplier

- Commands: `analyze-url`, `multiply`
- Inputs: URL, text, Markdown, transcript, simple local files
- Outputs: Markdown or JSON reports with multi-platform content drafts

### Prompt Generator Pro

- Command: `prompt`
- Inputs: Short creative or technical brief
- Outputs: Model recommendations and reusable prompt bundles

### GitHub Scout

- Command: `github`
- Inputs: GitHub URL or `owner/repo`
- Outputs: Repository health summary, release snapshot, issue sample, integration notes

### Social Publisher

- Planned command family: `social`
- Initial targets: Instagram Business / Creator, Facebook Page, LinkedIn, YouTube, X
- Workflow: draft -> policy checks -> human approval -> official API publish or schedule
- Hard rules: no fake-account creation, no password storage, no cookie automation
