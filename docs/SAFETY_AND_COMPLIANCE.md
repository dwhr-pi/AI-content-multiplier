# Sicherheits- und Rechtsnotizen

AI-ContentMultiplier erzeugt Content-Entwuerfe. Es ersetzt keine rechtliche,
medizinische, finanzielle oder professionelle Pruefung.

## Erlaubte Nutzung

- Eigene Inhalte umarbeiten.
- Inhalte mit Nutzungsrecht analysieren.
- Oeffentlich erreichbare Webseiten im Rahmen der jeweiligen Nutzungsbedingungen analysieren.
- Quellen fuer interne Notizen zusammenfassen.

## Nicht automatisch tun

- Fremde Inhalte ohne Recht republizieren.
- Login-geschuetzte Inhalte scrapen.
- Plattformregeln umgehen.
- Personenbezogene Daten ohne Grundlage verarbeiten.
- Automatisch posten.

## Cloud-Modelle

Cloud-Modelle koennen Daten an externe Anbieter senden. Deshalb gilt:

- Standard ist Ollama lokal.
- Cloud nur mit `ALLOW_CLOUD_MODE=true`.
- Sensible Quellen nur lokal verarbeiten.
- Ergebnisse kennzeichnen, wenn Cloud-Modelle verwendet wurden.
